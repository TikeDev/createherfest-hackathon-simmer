# Alarm Sounds

This directory contains built-in alarm sounds for the cooking timer.

## Built-in Sounds:
- `gentle.mp3` - Soft, calming chime (2-3 seconds)
- `moderate.mp3` - Medium intensity bell (3-4 seconds)
- `urgent.mp3` - Clear, attention-getting alarm (4-5 seconds)

## Specifications:
- Format: MP3 (best cross-browser support)
- Sample rate: 44.1kHz
- Bitrate: 128-192 kbps
- Duration: 2-5 seconds each
- Volume normalized to prevent clipping
