## **🔍 Source Type & License Guide**

### **Required Information by Type**

* **Data:** Include license (CC-BY, etc.), date accessed, and note if you removed PII (Personally Identifiable Information).  
* **Research:** Provide full citation (Author, Year, Title), DOI link, and fair use justification if applicable.  
* **Code:** List package name, version, license (MIT, Apache, etc.), and link to the repository.  
* **Visuals:** Name the creator/photographer and the specific license (Unsplash, Pexels, etc.).  
* **Quotes:** List the person’s name/role and confirm "Consent received".

### **License Quick-Check**

* ✅ **Permissive (Safe):** MIT, Apache 2.0, CC-BY, Unsplash.  
* ⚠️ **Restrictive (Use Caution):** GPL (requires your code to be open source), CC-BY-NC (Non-Commercial—prize money may conflict).  
* ❌ **Avoid:** "All Rights Reserved," Unknown sources, or images with watermarks.
